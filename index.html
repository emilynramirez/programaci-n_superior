Descripción del Proyecto

El objetivo principal del proyecto fue desarrollar un sistema capaz de detectar la madurez de frutas utilizando una cámara y técnicas de visión artificial, implementadas a través de la librería OpenCV.

Metodología

Para que la Raspberry Pi fuera capaz de reconocer objetos (en este caso, frutas) mediante una cámara, fue necesario implementar el uso de OpenCV, una librería de código abierto ampliamente utilizada para el procesamiento de imágenes y visión por computadora. Fue necesario instalar el paquete de OpenCV directamente en la Raspberry mediante los siguientes comandos en su terminal:

  • sudo apt update
  • sudo apt upgrade
  • sudo apt install python3-opencv

    • Una vez instalado el paquete de OpenCV y el código base de detección de madurez de frutas, se conectó una cámara web a la Raspberry para corroborar que pudiera reconocer las frutas y analizar su madurez. Posteriormente, se hicieron modificaciones en el código para que, cuando se detectara una fruta madura, un LED verde se encendiera simultáneamente; en caso contrario, se encendería un LED amarillo.

    Tras finalizar las modificaciones, se conectó la Raspberry a un protoboard para que sus señales llegaran a los LEDs mediante cables jumper. Se realizaron pruebas para verificar el funcionamiento del programa. Finalmente, se constató que la Raspberry fue capaz de procesar el código y analizar la madurez de las frutas con eficacia.
